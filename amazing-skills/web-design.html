Web Design

For synthesis two, The Evolution of Chance, I scaled each of my nine charts differently. This led to many similarly sized points representing completely different amounts. This was especially bad because I couldn’t even figure out how to create rollovers that would show the true amount.

However, I decided to continue with this because of my audience; I wanted this website to appeal to music lovers and artists. My main goal was to create a visually pleasing and engaging page. If this project were made for a more scientific audience, I would not be able to get away with it. In that case, I might have to use some other method to eliminate outliers, such as just ignoring them or only focusing on categories that didn’t have huge outliers.
