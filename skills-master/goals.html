Goals and Workplan

For my first project, I would like to make something like the college graduation speech analyzer you showed in class to analyze and compare Chance the Rapper’s lyrics over time. I would like to include links to the songs, or even embed SoundCloud players directly on the site. I also think it would be really cool to implement the Genius API, which analyzes and explains lyrics line by line.

This would be a fun project, but it would also showcase many skills that could easily be used in more serious work. I’m not sure how hard it would be to integrate that API, but if I pulled that off it would be really cool. Learning how to do that would all be work outside of class on my own.

Getting this project done would rely heavily on checking out the code for the original college graduation speech analyzer and other projects like it. I can’t remember what it’s called so I’ll start by asking you for the link. I would probably write this project in Python or Javascript.

I haven’t decided on a topic for my second project, but I really like the idea of creating a long infographic with lots of different simple interactive charts. I’ve designed several static infographics in the past, but have never made anything that viewers can actually interact with. Also, having many different charts would make up for the fact that some of the data may be incomplete.
